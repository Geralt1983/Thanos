# Commitment Entry Template

## Quick Add Format
```
- [ ] [What] | [To Whom] | [Due Date] | [Status]
  - Context: [Why this matters, any blockers]
  - Added: [Date]
```

## Full Entry Format

### What
[Specific deliverable - be concrete]

### To Whom
[Person or organization this is owed to]

### Due Date
[Specific date or timeframe]

### Status
- [ ] Not Started
- [ ] In Progress
- [ ] Blocked
- [ ] Complete

### Context
[Why this matters, background, any dependencies]

### Success Criteria
[How will you know it's done?]

### Blockers
[What could prevent completion?]

### Next Action
[The very next physical action to move this forward]

### Time Estimate
[How long do you think this will take?]

---

## Status Definitions

**Not Started:** Haven't begun work yet
**In Progress:** Actively working on it
**Blocked:** Waiting on something/someone
**Complete:** Delivered and confirmed

## Example Entries

### Work Example
```
- [ ] Baptist Rehab optimization status report | Baptist team | 2026-01-10 | In Progress
  - Context: Weekly update on Optimization project progress and blockers
  - Added: 2026-01-05
  - Next action: Pull metrics from system, draft key findings
```

### Personal Example
```
- [ ] Research home office conversion options | Self/Ashley | 2026-01-15 | Not Started
  - Context: Sunroom to office conversion for better work setup
  - Added: 2026-01-05
  - Next action: Get 2-3 contractor quotes
```
