# Personal Assistant Commands

Personal assistant commands for Thanos life management system.

## Available Commands

| Command | Description |
|---------|-------------|
| `/pa:daily` | Morning briefing with Oura data, tasks, and priorities |

## Usage

```bash
# Run morning briefing
/pa:daily

# Focus on specific area
/pa:daily epic
```

## Integration

These commands integrate with:
- Oura Ring (health/readiness data)
- WorkOS MCP (tasks, habits, energy tracking)
- State files (CurrentFocus.md, Commitments.md)
