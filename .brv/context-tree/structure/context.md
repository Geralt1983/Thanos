# Domain: structure

## Purpose
Defines the high-level architecture, core principles, and structural layers of the Thanos Operating System.

## Scope
Included in this domain:
- Core architectural principles (Backend-only, MCP-first)
- System layering (Orchestrator vs. Backend)
- Interface contracts (MCP, State files)

Excluded from this domain:
- Specific MCP tool implementation details
- Frontend component logic (WorkOS-v3)

## Ownership
Thanos Core Team
